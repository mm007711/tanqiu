# 锚链水手弹珠机 · 联机布场原型 v1

这个包体包含：

- `public/index.html`：完整可运行的游戏原型
- `server.js`：WebSocket 房间同步服务器
- `package.json`：Node 启动配置
- `docs/`：当前系统设计文档副本，如存在

## 本地单机 / 本地双人

直接打开 `public/index.html` 即可。

掌舵手操作：

- `SPACE`：蓄力发射
- `A / D`：调舵
- `X / SHIFT`：锚点瞄准 / 抛锚
- `1 / 2 / 3`：短链 / 重锚 / 回航锚

航海士布场：

- `4`：临时护航，消耗 2 航材
- `5`：导流斜架，消耗 2 航材
- `6`：破障爆板，消耗 3 航材
- `7`：航道调度 / 封港木板，消耗 3 航材

## 航材规则 v1

- 开局：3 / 8
- 每次新球开始：+2
- 每 6 combo：+1
- 补给口袋：+2
- 完美锚点：+1
- 潮汐狂涌：+2
- 压裂击碎：+1

## 压裂点规则 v1

木板被小球或重锚命中后累计局部压裂层数。

- 2.5 秒内再次命中：继续累计
- 超时：层数缓慢衰减
- 达到阈值：进入龟裂 / 临界
- 击碎后：
  - 获得爆分
  - 小球向上或沿斜板弹飞
  - 获得 5 秒内下一次锚链免耗

## 联机测试

需要 Node.js 18+。

```bash
npm install
npm start
```

然后打开两个浏览器窗口：

掌舵手：

```text
http://localhost:8787/?room=test&role=helm
```

航海士：

```text
http://localhost:8787/?room=test&role=navigator
```

当前联机模型：

- 掌舵手端是权威模拟端
- 航海士端按 `4 / 5 / 6 / 7` 发送布场请求
- 掌舵手端执行布场并广播快照
- 航海士端接收状态快照进行观战同步

这是可玩原型同步，不是最终商业级回滚 / 帧同步。
后续如果要做严肃在线对抗，建议继续拆分成：
`shared rules`、`authoritative server`、`input queue`、`snapshot reconciliation` 四层。
